# 1.HTML基础

# 基础认知

html、css、js实际上都是纯文本文件。纯文本文件，其内容只包含文字的文件，不包含文字样式。记事本txt文件是典型的纯文本文件。doc是富文本，不仅保存了文字内容，还保存了文字样式。所以，相同字数的doc文档比txt文件大。

HTML，hypertext markup language，超文本标记语言，是用来制作网页时，负责描述文档语义的语言。html通过超文本标记给普通文本添加浏览器可以识别并解析的语义，这些超文本不会在浏览器中显示，在编辑器中就是一些可以编辑的普通文本。（所有的超文本语言中的标签及符号，都是在英文状态下的）。

# HTML结构

基本骨架中，html标签是最根本的标签，表示整个网页；Head是网页的头部标签，主要放网页的配置，除了title标签外中的内容外，其他标签内容在浏览器中是不可见的；Body是网页的内容。

文档DTD，doctype definition，定义文档版本，使用哪个版本的规范。不同的版本，遵循不同的规范，对标签的用法和语法不同。对HTML、CSS、JS规范维护和制定的组织是W3C。

HTMl1.0-HTML5
HTML4.0版本，将结构和样式进行了分离；
常用的HTML4.01版本；
XHTML1.0在4.01版本基础进行严格化和扩展升级，严格规定标签必须小写，属性必须用双引号包裹，结束标签的关闭符号/必须写。
Strict严格版：
不能使用font等废弃标签，不能使用框架集，结构和样式分离；Transitional过渡版（通用版）：可以使用font等废弃标签，不能使用框架集；Frameset框架集版：可以使用框架集；
严格程度：
XHTML1.0 Strict> HTML4.01 Strict>XHTML1.0 transitional> HTML 4.01 transitional;
HTML5的DTD进行了极大的简化：<!DOCTYPE html>。发展到HTML5版本，进行了大幅度更新升级，取消了3个小版本

  • html、head、title、h1-h6、p、br、a、dl、dt、dd、ul、ol、li、div、a、img、button、input
