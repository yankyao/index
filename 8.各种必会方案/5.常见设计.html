# 5.常见设计

# 图片显示方案

图片是占用请求较大的资源。图片的load完成后，再整体显示。一张图片的加载是从上到下加载的，为避免出现图片加载一半的情况，提高用户体验（图片要么不显示，要么完整显示），图片在没有加载完成之前用小菊花代替；

// 加载完成事件：
img.onload = function(){}

实现加载完成在显示：

  1. 让父盒子带小菊花背景，img设置display：none；onload函数执行display：inlineblock；
  2. 不能给一个已经上树的元素加上load事件;
  • 先new Image()，设置src请求，此时即使不上树，图片的HTTP请求也会发出（图片的特点）；
  • 然后这个新的img对象就能够监听onload事件，onload之后:
  1. 上树new出来的img;
  2. 或者把图片地址赋值给已上树的img标签，会直接从缓存得到图片；
  • 用代理类批量操作，可以做懒加载/预加载等常见方案；
  • 可使用用占位符data-src；
const img = new Image();
img.src = 'https://**/**.png';
img.onload = function() {
    /* 上树或者把图片地址赋值给其他img */
}

# 瀑布流

其实现原理：①绝对定位；②谁先load完成谁先上树；③依次上树到最短列。

# 预加载

浏览器有一个特点，就是在页面不刷新的时候，对相同的URL不会发出多次请求。所以这就是预加载的实现原理，直接new出需要加载的图片提前发送图片请求，然后再需要的时候上树。

# 懒加载

看不见图片的时候，不加载，直到屏幕卷动快要看到它的时候在加载它；这样可有有效减少HTTP请求，保证第一屏的加载速度。

// 使用JQUERY
lazyLoad()
$(window).scroll(lazyLoad)
function lazyLoad() {
// 获取已卷动的高度
    var scrollHeight = $(window).scrollTop()
// 窗口高度
    var clientHeight = $(window).height()
    $('.lazy').each(function() {
// 懒加载图片的净高度
        var lazyItem = $(this).offset().top
        // 距离图片200px时进行加载
        if(scrollHeight + clientHeight > lazyItem - 200) {
            $(this).attr('src', $(this).attr('data-src'))
            $(this).removeClass('lazy')
        }
    })
}

进一步来限制加载的元素：当一个区域内所有的图片都需要加载完毕后，再集体显示；此时需要给这个大盒子加上lazy类名，遍历所有图片，设置data-src属性：

<!-- html -->
<div class="section section1 lazy">
    <img data-src='imgaes/1.jpg' class='lazy'>
    <img data-src='imgaes/2.jpg' class='lazy'>
    <img data-src='imgaes/3.jpg' class='lazy'>
    <img data-src='imgaes/4.jpg' class='lazy'>
</div>
<div class="section section2 lazy">
    <img data-src='imgaes/11.jpg' class='lazy'>
    <img data-src='imgaes/22.jpg' class='lazy'>
    <img data-src='imgaes/33.jpg' class='lazy'>
    <img data-src='imgaes/44.jpg' class='lazy'>
</div>

/* js */
lazyLoad();
$(window).scroll(lazyLoad)
function lazyLoad() {
    // 获取已卷动的高度
    var scrollHeight = $(window).scrollTop()
    // 窗口高度
    var clientHeight = $(window).height()
    $('div.lazy').each(function() {
        // 懒加载图片的净高度
        var lazyItem = $(this).offset().top
        // 距离图片200px时进行加载
        if(scrollHeight + clientHeight > lazyItem - 200) {
            // 备份dom
            var $dom = $($(this).html())
            // 清空
            $(this).empty()
            // 放入loading图片
            $(this).append("<img src='loadng.gif' class='loading'/>")
            // 备份
            var $self = $(this)
            // 当前区域图片数量
            var amount = $dom.find('img.lazy').length
            // 计数
            var count = 0
            $dom.find('img.lazy').each(function() {
                $(this).attr('src', $(this).attr('data-src'))
                $(this).removeClass('lazy')
                $(this).load(function() {
                    count++
                    if(count === amount) {
                        $self.empty().append($dom)
                        $self.removeClass('lazy')
                    }
                })
            })
        }
    })
}

# 倒计时秒杀

做倒计时秒杀，用的是服务器时间：任何一个HTTP的下行response头部都会携带服务器的时间；responseHeader我们只能get不能set，因为set是服务器的事情。

xhr.getResponseHeader('Date')

在页面开始的时候请求一次服务器时间；计算页面开始的这一个瞬间（T0），与目标时间(T1)的毫秒差；用此时时间（T2）与T0的差，来修正页面的时刻与目标时间之间。

var serverDate = null
var xhr = new XMLHttpRequest()
xhr.onreadystatechange = function() {
// readyState为3时，就可以获取服务器时间了
    if(xhr.readyState === 3) {
        var date = xhr.getResponseHeader('Date')
        serverDate = new Date(date)
        timeStart()
    }
}
// 使用Math.random设置访问戳，避免相同请求被浏览器拦截
xhr.open('get', 'xfgd.php?' + Math.random(), true)
xhr.send(null)
function timeStart() {
    var i = 0
    var targetDate = new Date('Dec 12 2018 00:00:00')
    setInterval(function() {
        i++
        // 注意修正8小时，北京时区为东八区，比格林尼治事件快8小时
        var haomiaocha = targetDate - serverDate - i * 1000 - 8 * 60 * 60 * 1000
        var miaocha = haomiaocha / 1000
        var fencha = miaocha / 60
        var shicha = fencha / 60
        var tiancha = shicha / 24    
        // 注意取整
        document.getElementById('time').innerHTML = '距离秒杀开始还有' + parseInt(tiancha) + '天'
        + parseInt(shicha % 24) + '小时' + parseInt(fencha % 60) + '分' + parseInt(miaocha % 60)
        + '秒'
    })
}
