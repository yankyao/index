# 6.性能优化

# HTTP层面

  • DNS 解析: 采用DNS Prefetch 一种DNS 预解析技术；
// 只有部分浏览器支持
<link rel="dns-prefetch" href="www.baidu.com" />
  • TCP 连接：采用http2.0，可以复用tcp通道，采用二进制格式而非文本格式，使用报头压缩；
  • 浏览器并发：对于某些静态资源，图片等等，我们可以对其URL分散处理 ，不同的资源域名；
  • HTTP请求次数：将多个请求合并成同一个；
  • HTTP压缩：开启Gzip压缩；
