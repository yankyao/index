# 1.基础概念

# Web及其标准

  • 广义：（World Wide Web）即全球广域网，也称为万维网，它是一种基于超文本和HTTP的、全球性的、动态交互的、跨平台的分布式图形信息系统。

  • 狭义：网站，包括web前端和web后端系统。

Web标准：由万维网联盟（W3C）制定。主要是用来统一浏览器厂商对web的规范。

# 前端三层

前端三层是Web标准的内容：

  • HTML控制构建网页的基本结构；
  • CSS控制设计网页的表现效果；
  • JavaScript控制开发网页的交互效果。

# 浏览器

用来获取和展示Web信息资源的应用程序。

Web信息资源使用统一资源标识符( Uniform Resource Identifier，URI)作为标记。

可以是网页、图片、视频、以及任何在Web上所呈现的内容。使用者可以借助超链接(Hyperlinks)，通过浏览器浏览互相关联的信息。

主要有三种内核：

  • WebKit内核：Safari、Chrome；
  • Trident内核：IE；
  • Gecko内核：Firefox；

浏览器组成：

  1. 用户界面；
  2. 浏览器引擎；
  3. 渲染引擎：html、css；
  4. 网络；
  5. UI后端；
  6. JS解释器：js;
  7. 数据存储；

# 服务器

狭义：传统服务器就是计算机，性能优于普通计算机。云服务器是硬件资源集约化然后进行分割出租，有强大的动态功能，降低了计算成本和管理成本。

Web服务器包含狭义的硬件和服务器软件，主要有以下：

从服务器软件种类分：

  • Apache；
  • IIS；
  • Nginx；
  • Tomcat；
  • Nodejs；

按照项目阶段：

  • development 开发环境；
  • test 测试环境；
  • release 预发布环境；
  • production 生产环境；

按照用途：

  • 静态文件服务器；
  • 数据库服务器；
  • 邮件服务器；
  • web服务器；

按照操作系统：

  • Linux;
  • Windows
