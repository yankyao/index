# Git

Git是一个版本控制、内容管理、工作管理的分布式系统。按元数据方式存储、内容存储使用SHA-1哈希算法。 基础操作：

git init // 创建新仓库
git clone // 创建本地仓库的克隆版本
git add // 添加到暂存区
git commit -m "代码提交信息" // 实际提交改动
git push origin // 提交到远端仓库
git checkout -b feature_x // 创建一个叫做feature_x的分支，并切换过去
git branch -d feature_x // 删掉feature_x分支
git push origin feature_x // 提交到远端仓库
git pull // 更新
git merge <branch> // 合并其他分支到当前分支
git diff <source_branch> \<target_branch> // 预览差异
git tag 1.0.0 1b2e1d63ff // 以1b2e1d63ff为点创建1.0.0 的标签
git log // 获取commit ID

git rebase：

// 合并提交记录
git rebase -i HEAD~2 // 之后p/s相应commit id。然后保存推出。
// 分支合并
// master改动领先本地dev
git rebase master // dev 分支是基于 hotfix 合并后的 master。且没有 merge 的 commit 记录

git push origin master --force // 强制覆盖远程分支

git reset:

// 合并提交记录
git reset --soft HEAD~1
git commit --amend // 修改commit message

# Charles

Charles其实是一款代理服务器，通过成为电脑或者浏览器的代理，然后截取请求和请求结果达到分析抓包的目的。

抓包手机：

  1. Proxy->Proxy Settings，设置端口号。
  2. 手机使用同一局域网，配置HTTP代理IP，为电脑的IP，以及相应的端口号。
