# 3.混合APP

# 概念

现在的手机中操作系统主要是Android、iOS、Window Phone，三个平台都选择了不同的语言，手机的系统中只能接受这种语言开发的东西。和Windows、linux不一样。windows中，开发语言可以各式各样的，QQ是C++写的，Webstorm是Java写的。手机系统不一样，比如iOS中，必须用swfit、OC来开发APP，不支持其他语言的。

公司学习成本、用人成本惊人的高。比如携程网APP团队，至少三拨人，安卓工程师、iOS工程师、WP工程师。里面还必须有自己的测试人员、运维人员、数据库人员等。很早的时候，大家就开始研究用WEB技术来开发APP。APP中都能防止web view组件，内嵌一个web页面进去，所以为什么整个APP不能就是一个网页呢？我们现在只需要写一份代码HTML、CSS、JS，就能部署到各个平台去，此时人员成本极大减少！

内嵌web view有问题：调用不了系统硬件API，比如：摄像头、蜂鸣器、陀螺仪、红外线、振动器、地理定位，你必须用Java、OC、.net原生语言做个壳子，通过bridge搭桥，来调用系统硬件的API。

# Cordova、Ionic

  Ionic是离子的意思，将两个大公司的大东西绑定在一起了：Google的Angular，Apache的Cordova。Ionic1对应Angular1，语言是js，内置webpack、less编译器；Ionic2对应Angular2，语言是ts，内置webpack，scss编译器。
