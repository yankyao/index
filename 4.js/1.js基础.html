# 1.js基础

# 简介

由Netscape网景公司，布兰登 艾奇实现。主要的作用：

  • 数据验证；
  • 读写html；
  • 与浏览器窗口极其内容交互效果；
  • 网页特效；
  • web游戏的制作；
  • 基于Node.js技术的进行服务器端编程。

完整的JS实现由三部分组成：语言核心ECMAScript、文档对象模型DOM、浏览器对象模型BOM。

# Hello World

可以放在HTML文件的任何位置，JS代码需要写在script标签内部，JS文件也是纯文本文件；

<script type="text/javascript">
    这是一段JS代码
<script>
  • alert()，JS内置的功能函数。可以让页面弹出一个警示框；alert语句写在script标签内部，是JS内置的功能函数，后面是英文状态下的小括号。括号里面写警示内容，内容如果是一段文字，必须用引号包裹，单双引号均可；
  • prompt()，JS内置的功能函数，页面会弹出一个对话框，可以让用户输入内容。prompt后也是跟英文状态下的小括号，里面可以写两个参数，中间用都好隔开，也可以只写第一个参数；用户输入的内容的是字符串类型的数据，返回到html中。
  • console()，JS内置的功能函数，可以在控制台的console标签栏中输出内容。

TIP

  • JS变量严格区分大小写，首个字符为字母、下划线、$，并且不能是JS的关键字和保留字;
  • html中xhtml规定用小写，其余不区分；
  • css不区分大小写，ID和类名区分；每条语句后面加分号（如果代码正好需要正常换行，也不会出错）；

JS代码，如果没有特殊结构，都是按照从上到下、从左到右的顺序依次加载；JS语句对换行、空格、缩进不敏感。

# 字面量

字面量是固定值的表示方法，也叫做常量。数字、字符串、数组、json等数据类型都有自己的字面量表示方法，即直接书写出。

数字字面量，分为整数、浮点数（即小数）、特殊值：

  • 整数字面量，JS常用的整数字面量是十进制、八进制、十六进制:
    • 八进制，前导符号0、0o、0O其一（注意八进制里只能出现0-7；超过了会转成10进制或者报错）；
    • 十六进制，前导符号0x、0X其一（0-9；a-f或者A-F。字母部分超过了，会直接报错）。
  • 浮点数字面量：JS中浮点数都是在十进制的模式下，重点记忆科学记数法的e。
  • 特殊值：无穷infinity、NaN。无穷分为正无穷和负无穷（直接写infinity或者-infinity；有一些算式也等于无穷，比如12/0；）；NaN，不是数学意义上的数字，例如0/0。NaN是Number类型的数据。

字符串字面量，包括空字符串和非空字符串，由单引号或双引号包裹；特殊字符：\n 换行、\t Tab制表、 ' 单引号、 " 双引号、 \ 反斜杠。

console.log('hello\nworld')

# 变量

变量，其本身相当于一个容器，可以指向任意类型的数据，字符串、数字、布尔值、数组、变量、函数等等。

  • 变量内部只能放置一种数据，每次被赋值，旧的数据就会被舍弃，用的时候用的都是新的数据( 理解这一点非常重要，再循环里特别要注意！)；
  • 原生JS是动态的，弱类型的语言，指的是变量的类型是根据变量的值来决定的。

变量声明的关键字是var(ES6的let、const) ，变量后面跟等号；变量一般先声明再赋值，赋值的符号是=。

TIP

  • 一个变量的每次赋值都会覆盖掉前面给变量赋的值；多个变量声明赋值用逗号隔开。

  • 变量声明的提升：变量的声明可以在输出后面，但是变量的赋值要在输出语句前面，否则变量的值为 undefined。

引用类型之判断的是"指针"是否相同，所以：

[1] != [2];
{a: 1} != {a: 1}
var a = [];

// a 不等于 b ，并且[ ]！=[ ];
a = [];

// [ ]不等于[ ]，相当于给a赋了两次值;

数字字面量和盛放数字字面量的变量都是number类型，所有的字符串字面量都是string;

变量的类型是其内部存储的数据的数据类型；

typeof 1; // 'numnber'
typeof(1); // 'numnber'
typeof(null); // 'object'
Object.prototype.toString.call(1); // "[object Number]"
Object.prototype.toString.call(null); // "[object Null]"

typeof(null)返回’object'，不够准确。Object.prototype.toString.call()，比较准确。

# 数据类型的转换

不同数据类型之间可以相互转化。

  • 数字转字符串，涉及加法运算的数字和字符串的表达式，JS会自动把数字转换为字符串。

  • 字符串转数字：Number()、parseInt()、parseFloat()，可以把非数值转为数值。

    • Number()可以转换任意数据类型，后面两个专门用于字符串/进制转换。
    • 使用parseInt将字符串转化为数字，字符串的开头是数字并且默认字符串里的数字为10进制，则会转为数字，其余的转为NaN；例如"021"→21、"x1010"→NaN、"10"→10、021→17；"5.2e10"→5.2。

TIP

parseInt，以规定进制，其默认为十进制：

  • 可以对数字进行取整运算；
  • 给数字转换进制，一般先把数字转换为10进制；
  • 将字符串转为数字，里面可以规定进制，其默认为十进制。
parseInt（021）         // 17（八进制转为10进制）
parseInt（021，8）      // 15（先把八进制转为10进制，再把得到的结果当作8进制数字，再转为10进制）
parseInt（'021'，8）.   // 17
parseFloat('021')      //  21
parseInt（"你好"）      // NaN
parseInt（"10你好"）    // 10
parseInt（Infinity）    // NaN；
]parseFloat（Infinity） // Infinity
parseFloat（".5"）      // 0.5
parseFloat（.5）        // 0.5

# 运算符

数学运算符，加+、减-、乘*、除/、取余%、小括号（小括号也是运算符）;

Math对象是JS内置的强大的数学对象，有很多属性和方法：

Math.random()         // 返回一个大于0小于1的随机数，出现随机性，既考虑random；
Math.pow(num,power)   // 返回num的power次幂
Math.sqrt(num)        // 返回num的平方根
Math.PI               // 返回π的值

TIP

Math.min(a ,b, c, d)和Math.max(a, b, c, d)可以取最大/小值。 在进行阀值判断时可以使用，减少代码，例如，当x小于0时取0：Math.max(x, 0)；

  • 比较运算符，Comparison operators，大于小于、大于等于 小于等于、相等不相等、全等不全等；
  • 逻辑运算符，Logical operators，或 || 、与（且）&&、非！；
  • 赋值运算符，Assignment operators，等于=、加等于+=、*=、/=、%=、++递加1、--递减1；
  • 条件运算符/三元运算符 Conditional operator: Boolean_expression ? true_value : false_value；

TIP

添加、获取属性的点 . ，也是运算符，且优先级最高；

# 运算符优先级

数学运算符，先乘除，后加减，有小括号先算小括号；正常的是数字与数字之间的数学运算；非常规：

  • 数字与纯数字字符串、纯数字字符串与纯数字字符串之间，除了加号外，其他运算会将字符串自动转为数字。这个过程叫做"隐式转换"，不需要parseInt等方法；
  • 其他类型的数据之间进行数学运算的时候，也会进行隐式转换。空字符串、null、false会转换成数字0；true会转换成数字1；NaN、undefined、非空非纯字符串转换成NaN。（NaN参与的数学运算结果都会返回NaN ）Infinity参与的运算情况有很多；

比较运算符，比较运算符输出的结果都是布尔值，要么是true，要么是false；常规的是数字与数字进行比较；非常规:

  • 数字与其他数据类型比较，其它类型会发生隐式转换为数字后，再进行比较：纯数字字符串会转换为数字，空字符串、null、false转换为0，true 转换为1；
  • 字符串与字符串进行比较，按照字符编码顺序，前面的小于后面的：字符串与字符串之间也能进行比较，注意二者必须都是字符串中的数字，下面的顺序；字符编码顺序：0-1，A-Z，a-z；比较方法是一个一个进行比较，而不是比较字符长度；
  • NaN除了不等于和不全等NaN 与返回true，其他参与的比较运算，都返回false；Infinity：
Infinity > Infinity    // false
Infinity < Infinity    // false
Infinity <= Infinity   // true
Infinity >= Infinity   // true
Infinity == Infinity   // true
Infinity != Infinity   // false
Infinity !== Infinity  // false
Infinity === Infinity  // true
  • 进行比较运算时，一个运算符就执行依次：例如：3>2>1；其比较结果为false：代码在执行的时候，先比较3>2，输出结果true；在比较true>1，则输出结果false；(Number(true) = 1; Number(false) = 0)

逻辑运算符，逻辑运算符常用于布尔类型值之间；当操作为布尔值时，返回值也是布尔值，如果将其他数据隐式转换为布尔值，计算之后，输出对应位置的值，不一定是布尔值。非零数字、非空字符串，会隐式转换为true；0、null、undefined、空字符串、null则会转换为false。

  • 逻辑与运算，与&& ：一假为假，全真才真；（JS代码在计算机运算中，是从左往右的顺序），因此，要使a&&b为真中第一值a为假时，就能判断a&&b整体为假，不需要看b，返回值是a(假)；第一值a为真时，不能判断a&&b整体是否为真，需要根据b的值来判断整体是否为真，返回b的值；
  • 逻辑或运算， 或||：一真为真，同假才假；
  • 逻辑非运算，非真即假，非假即真；

WARNING

赋值运算符，即先进行运算，再进行赋值。可以看出，赋值运算是有变量参与的运算；特殊符号++，可以写在变量前面，也可以写在后面；a++第一次使用的是原来的值，再次使用则是加完后的值（后面的a都等于原来的a加上1）；++a第一次使用就是加完后的值，后面再次使用也是加完后的值；

//注意--a和a--区别的
let a = 10
console.log(a--) // 10
console.log(a) // 9

let b = 10
console.log(--b) // 9
console.log(b) // 9

TIP

综合运算顺序，从左往右：

  1. .运算；
  2. ++/--/!；
  3. 数学运算；
  4. 比较运算；
  5. 逻辑运算中的（先算&&，再算 ||）
  6. 赋值运算

# 对象

  • 数组存储的是一组数据，可以通过index获取数据。
  • 对象中，每个数据都有自己的key。可以通过数据的名字得到数据的value。

TIP

  • 数据多的时候，数组的使用效率变低。
  • 注意在运算的时候数组和对象使用的取舍；
  • 使用obj[key]获取某key对应的value，key会被计算成字符串；
  • 因为点语法也是一步运算，所以多次引用引用类型值中的某一项，应该赋值给变量，变量存在栈内存中。

对象字面量语法：{}内部包括多个数据，每个数据之间用逗号隔开，最后一个数据后不能写逗号。每一项数据都包含属性名和属性值，键值对写法k:v。

const obj = {a: 1};
/* 修改 */
obj.a = 2;
/* 删除 */
delete obj.a;
/* 添加 */
obj.b = 3;

for(var item in obj) {
    console.log(item, obj[item])  // b, 1
}

TIP

引用类型值的复制，由于JSON对象是引用类型值；所以复制的时候必须考虑是深复制还是浅复制。

Object.assign({}, {a: 1}) // 浅复制

loadsh.deepClone // 深复制

使用JSON.stringfy可以某些情况下检验复杂对象是否完全相等。

# JS中的声明提升

"函数会首先被提升，然后才是变量"。函数不存在赋值流程，一个完整的变量生成分为声明和赋值两个步骤，变量声明并赋值只会提升声明；不声明只赋值不会提升； 函数提升优先级比变量提升要高，且不会被变量声明覆盖。

foo() // 'world'
var foo = 'hello'
function foo() {
    console.log('world')
}
var foo = 'hello world';
foo = 'hello world';

但是后面会被变量赋值覆盖，因为变量的赋值流程是从上往下的，不会出现提升；

function foo(){
    console.log("函数声明");
}
var foo;
console.log(foo);  // 输出上面函数
foo = "变量";
console.log(foo);  // '变量'
