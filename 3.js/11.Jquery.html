# 11.Jquery

# 简介

  jQuery不是一门新的语言，只是一个原生JS书写的库；简化了DOM编程，封装了很多方法：

  • 操作HTML的DOM元素，获得元素、获得属性、添加、删除节点；
  • 操作css样式，更改和添加css属性（也是行内样式）；
  • 添加事件监听；
  • 综合运用的一些页面特效；

  jQuery就是为了简化DOM而产生。做了大量的兼容工作，即使是IE6也是全部兼容。   引用位置，jQuery简化了语法，但是没简化业务和逻辑；就是外部的JS库：

<script type="text/javascript" src="js/jquery-1.12.3.min.js"></script>

# $对象

  jQuery对象，通过$()获得的就是jQuery对象。我们所有的方法和属性基本都是先调用对象。$()其实是jQuery这个库给我们提供的一个唯一的全局变量。这是jQuery比较好的一点，不会造成全局的污染。

  最开始不是写作$，写作jQuery()。库里两个名字是共同存在。作用相同。通过$（）获得的就是一个jQuery的对象，封装大量的自己的属性和方法，.html()和.css()和animate()都是jQuery对象内部的方法。jQuery对象只能调用jQuery的方法和属性，不能调用原生js的属性和方法，原生JS也不能使用JQuery的方法；

  $()获得的元素是一个类数组的JQuery对象：$().length属性，可以返回内部元素的长度。可以通过下标获取某一项元素对象；$().size()方法，可以返回我们内部元素的长度；$().index()方法，能够返回元素在结构中同级元素中的位置，下标。   原生js对象和jQuery对象之间是可以互相转化的：

// jQuery对象→原生js对象：方法就是根据下标获得其中的一项元素。
$("p")[0].html("你好");//会报错
$("p")[0].innerHTML = "nihao"
// 原生js对象→jQuery对象：将原生js对象$()一下就可以了。
var box = document.getElementsByTagName("div")[0]
$(box).html("nihao")

  jQuery的方法都是在jQuery对象里面，只有一个$()方法获得jQuery对象，我们的jQuery程序所有语句都必须以$()。

# 选择器

  比较基本的JQuery的选择器：

$('p')            //选择所有p标签元素
$('.box')         // 选择类名为box的元素
$('#demo')        // 选择ID为demo的元素
$('div.box')      // 选择（所有div）中类名为box的元素
$('div ul li p span.btn')
$('p,ul,div')     // 选择所有的p、ul、div标签；
$('*') // 选择整个页面；
$("input[name='']") // 选择name为的input控件；

  进一步筛选选择器：

$("p").css("background","red")          // 所有p标签
$("p:first").css("background","red")    // 第一个p标签元素
$("p:last").css("background","red")     // 最后一个p标签元素
$("p:eq(2)").css("background","red")    // 下标为2的p元素（下标从0开始）
$("p:gt(2)").css("background","red")    // 下标大于2的p元素
$("p:lt(2)").css("background","red")    // 下标小于2的p元素
$("p:odd").css("background","red")      // 下标为奇数
$("p:even").css("background","red")     // 下标为偶数

# jQuery的常用方法

  html（），相当于原生JS中的innerHTML方法：给元素内部添加内容；（必须用双引号包裹）；不仅可以添加直接显示的内容，还可以给元素内部添加标签（创建节点）：

$("div").html("<p><a href=\"#\">你好</a></p>")
// 读取内容（只能读取第一个元素内的内容）：
$("div").html()

  css（），调用和更改CSS样式。设置一个属性可以传递两个参数：如果只传一个参数，调用属性计算后的样式，如果传递两个参数，就是修改这个样式。jQuery优化的很好：属性值很灵活，可以直接写数字不带单位，还可以写带单位或不带单位的字符串；属性名既可以写驼峰也可以写css写法；设置多个属性，将多个属性写在JSON对象。   addClass（）和removeClass（）,toggleClass（），增加类名和移出类名；   attr（），注意此处比原生相区别，原生直接调用即可。调用或更改html元素的属性和属性值，相当于原生JS中的直接打点。可以传两个参数修改：一个是属性名，一个是属性值。只传一个属性名，就是获得这个属性的属性值。移出自定义属性：removeAttr(“属性名”)。   常用的事件，JQuery直接给类数组对象批量添加事件，$（this）则指的是触发事件的对象。

// jQuery不需要写on，事件函数写在括号里；
click() //单击事件
mouseenter() // 鼠标移上，鼠标进入；
mouseleave() // 鼠标移出，鼠标离开方法；
mousemove() // 鼠标移动
input() // 输入事件
// 还有一种写法
// 第一个是时间时间类型，第二个是事件函数；
$().on("click", fun)

  hide（）、show（）、toggle（），就是隐藏和显示某一个元素；可以传递一个时间参数，控制隐藏和显示的中间时间；伴随着透明度从设置透明度到0的变化，还有宽高从设置值到0的动画。

  slideDown（）、slideUp（）、slideToggle（）。fadeIn（）和fadeOut（）、fadeTo（）、fadeToggle（），表示淡入和淡出：透明度发生的变化；fadeTo（）。有两个参数，第一个叫做速度参数（“slow”、“fast”等）。第二个透明度的终点。

# 节点关系/选择器

  jQuery控制元素时，都是批量控制。添加事件，元素本身运动和其他元素运动，需要跟事件元素找一些关系。jQuery帮我们简化了这个找关系的过程，封装了一系列的方法可以帮我们找到自己、父亲、儿子、兄弟、祖先、孙子等元素。

  • $(this)自己，事件内部有一个$(this)的对象，指向的就是触发事件的这个元素本身。就是原生js里的事件函数内的那个 this，用$()将this转成jQuery对象，就能调用jQuery的方法。this不要加引号，一般写在事件函数或者each函数里。
  • parent（）父亲，寻找元素父级的方法：必须加小括号，找到的也是亲生的父亲。爷爷和叔叔都不能。方法返回的是某一个元素的父亲的jQuery对象，继续打点调用jQuery方法。
  • children（）儿子，寻找子级元素的方法：必须加小括号，选中的是元素的所有儿子级元素。孙子级的元素选不中；方法可以传递参数，指的是我要选择儿子级里面还要满足选择器要求的那些元素。
  • siblings（）兄弟，寻找的是同级同父元素的兄弟元素：方法必须加小括号，找到的是亲兄弟元素；以通过参数进行选择器筛选。
  • parents()，寻找祖先元素，父级、父级的父级，可以通过参数进行选择器筛选。

  连续打点调用，原理：一个jQuery对象打点调用完自己的任何方法，都会return一个执行完该方法的jquery对象，选择器中一般是选中的元素。其他的节点关系：

find() // 找的是元素的后代的一些元素，根据参数的选择器去选择；加逗号可以多选；
next() // 选中的是当前元素的下一个元素，必须是同级的；
nextAll() // 选中的是当前元素后面的所有兄弟元素。内部可以传参数，选择符合选择器的后面的兄弟；
prev() // 选中的是当前元素的上一个元素，必须是同级的；
prevAll() // 选中的是当前元素前面的所有兄弟元素。内部可以传参数，选择符合选择器的后面的兄弟；
parents() // 选择的是包含html标签在内的所有祖先元素。内部可以传参数，选择符合选择器的祖先元素；

# jQuery遍历DOM

  eq( )筛选，通过$（）获取的jQuery对象是一个类数组对象，可以继续通过.eq（）来获取某一个元素，下标从0开始。

  index（）筛选，获取的是这个元素在html中，在自己同级元素中的位置；

  对应和排他。对应：通过兄弟关系的元素位置来对应；排他：与己对应设置，其余的恢复默认；

  each（）遍历jQuery对象，each(index,element)可以把触发事件的元素自己的下标（在元素类数组中的），当作参数传进去。与常与$(this)一起用。

# animate动画

  原理是定时器，改变步长和总时间时间的关系。   动画队列，同一个元素身上设置多个动画，则会出现动画排队；不同元素不会出现排队；delay（）延迟动画:

$('.demo2').delay(2000).animate({'left': 600, 'top': 400}, during, function(){
    $('.demo2 p').css('display', 'block')
})

  stop（）停止动画，可以传两个参数，都是必须是布尔值；第一个参数表示：是否清空排队的动画，true表示清空，false表示不清空；第二个参数表示：是否立即完成当前动画，true表示立即完成，false表示立即停止。默认不写表示两个都是false；只写一个true表示两个都是true。

  解决动画排队问题。

  • stop方法：为了防止频繁流氓操作，只要添加一个运动动画，前面都必须加一个stop(true)。清空前面所有动画队列，立即停止当前动画。参数值需要传第一个true；
  • 节流方法：判断元素是否在运动过程中，如果是，就不执行后面的其他操作，如果不是，就执行后面的动画，元素都有一个方法叫做is()，判断是否处于某种状态。
$('p').is(':animated') // 返回布尔值

$("<li>"+"要添加的内容"+"</li>").appendTo($('div'))    // 添加到节点前面
$("<li>"+"要添加的内容"+"</li>").prependTo($('div'))   // 添加到节点后面
