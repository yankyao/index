# 2.数组字符串

# 概述

数组是引用类型值，是一个对象型数据(object)。一组有序的数据集合，一个数组存放一组数据；组内可以有不同的数据类型。 创建数组字面量用中括号[]，用变量存放数组指针/地址、使用数组。

数组有index属性、length属性，index是数组内每个数字的下标，从0开始；调用时，如果超过数据的个数，会显示未定义项(undefined)；可以直接通过下标更改数组内对应的数据。

# 数组操作方法

  打点调用方法。   首尾操作，是在操作原数组。

push()        // 末尾添加元素，在原数组操作          // 改变原数组
pop()         // 移出并返回最后一个元素          // 改变原数组
shift()       // 移出并返回第一个元素          // 改变原数组
unshift()     // 开头添加元素，在原数组操作          // 改变原数组

合并、拆分。

concat()// 合并数组，返回新数组；
slice(start，end)，        // end>start，提取片段，不包含end位置；可以为负，表示从最后往前数
splice(index，数量)        // 删除          // 改变原数组
splice(index，0，元素)     // 插入          // 改变原数组
splice(index，index，元素) // 替换          // 改变原数组

js倒序和排序，改变原数组；

reverse() // 倒序          // 改变原数组
sort()  // 不传参数，按照字符编码顺序排序(数字，大写字母，小写字母，依次比较)；传参数比较大小，则小在前大在后(改变原数组)

TIP

  • 注意某些方法是否会改变原数组；
  • 当前业务场景是否需要改变原数组；

  转字符串。

join() // 转字符串，不传参数则以逗号连接；传参数，即连字符

# 字符串

  length属性，同数组   常用方法

charAt()                // 可返回指定位置的字符串，index值也是从0开始；这是字符串的遍历方法
concat()                // 连接两个或者多个字符串
indexOf()               // 可返回指定字符串值在字符串中首次出现的位置，得到的是索引值
lastIndexOf()           // 可返回指定字符串值在字符串中最后出现的位置
slice()                 // 提取字符串片段
split()                 // 将字符串分割成数组
toUpperCase()           // 字符串转为大写；打点使用，没有参数；输出新数组
toLowerCase()           // 字符串转为小写；打点使用，没有参数；输出新数组
substr(start，length)   // start可以为正可以为负，从start开始截取length个字符串
substring(index，index) // index不可以为负；但是前面可以比后面大，JS在提取之前会调换位置
